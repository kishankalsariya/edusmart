# EduSmart - Student Management System

A full stack Student Management System built to manage 
student records, attendance, and academic performance.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Tool:** XAMPP

## ✨ Features
- Student Registration & Login
- Attendance Tracking
- Marks & Grade Management
- Admin Dashboard
- CRUD Operations on Student Data

## 🚀 How to Run
1. Install XAMPP
2. Clone this repository into `C:\xampp\htdocs\edusmart`
3. Import the database from `/database` folder into phpMyAdmin
4. Open browser and go to `http://localhost/edusmart`
